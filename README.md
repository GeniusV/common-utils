## About
This project contains some components which can be reused in many quick-start situtations.

Currently, This project includes:

- shiro
    + shiro listener
    + shiro redis cache
    + shiro session
- jedis
    + quick-start jedis dao
- utils
    + LoggerUtil, use Log4j
    + SerializeUtil
